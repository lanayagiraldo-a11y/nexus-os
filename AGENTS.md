## 🧠 Work Log — Regla obligatoria para TODOS los agentes

Cada vez que un agente inicia una sesión en este repositorio:

1. **LEER** `WORK_LOG.md` — para saber qué está pasando y qué se hizo antes
2. **AL TERMINAR** una tarea — agregar entrada al final de `WORK_LOG.md` con:
   - Fecha y hora
   - Nombre del agente
   - Tarea realizada
   - Estado (✅ completado / 🔄 en progreso / ❌ bloqueado)
   - Ubicación de archivos generados

3. **Estructura de carpetas** para guardar outputs (crear si no existen):
   ```
   proyectos/la-carolina/     → datos, reportes, dashboards
   proyectos/iera/            → reportes donantes, contenido
   proyectos/el-salvador/     → Dar Ibrahim, fondo inversión
   proyectos/taqwa-team/      → consultoría marketing
   proyectos/personal/        → arte, salud, familia
   ```
